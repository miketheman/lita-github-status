lita-github-status

A Lita handler to display current GitHub Status.

Installation

Add lita-github-status to your Lita instance's Gemfile:

gem 'lita-github-status'

Configuration

None

Usage

Ask the bot how GitHub is doing:

Lita github status
Status: good (300 seconds ago)

See Lita: help github for all commands.

Prior Art

Pretty much a direct port of Hubot's GitHub Status plugin.
